# 🧠 Synapse

Synapse is a modern desktop-based chat interface built using Electron, designed to deliver a fast, responsive, and visually clean messaging experience. Inspired by neural connections, Synapse focuses on seamless interaction, smooth performance, and a minimal user interface.

---

## 🚀 Features

- 💬 Clean and modern chat UI
- ⚡ Fast and responsive desktop performance
- 🖥️ Built as a cross-platform desktop application using Electron
- 🎨 Minimal dark-themed interface
- 🚀 Optimized rendering with GPU acceleration flags

---

## 🛠️ Tech Stack

- **Electron** – Desktop application framework
- **HTML5** – Structure
- **CSS3** – Styling (Dark UI)
- **JavaScript** – Application logic

---

## 📂 Project Structure

```
synapse/
│── index.html       # Main UI
│── main.js          # Electron main process
│── package.json     # Project configuration
│── node_modules/    # Dependencies
```

---

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/synapse.git
cd synapse
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Run the App

```bash
npm start
```

---

## 🧪 Build (Optional)

To build the app:

```bash
npm run build
```
